COME AVVIARE L'APPLICAZIONE

Per avviare ArtN7 seguire i seguenti passi:

- Aprire un terminale
- Posizionarsi nella cartella del software
- Rendere eseguibile il file startArtN7.sh con il comando:
    "chmod +x startArtN7.sh"
- Avviare l'applicazione con il comando:
    "./startArtN7.sh"

-------------------------------------------------------------------------------------

COME ACCEDERE ALL'APPLICAZIONE

Per accedere ad ArtN7 aprire un browser all'indirizzo "http://localhost:8008".
