How to Delete a Single File of a Catalog Title

This action will delete the selected files from both the database and the filesystem.

  1. Navigate to the Files section.
  2. Select the action "Delete this file from the catalog".
  3. Check the files you wish to delete.
  4. Click on Go.

How to Delete an Entire Catalog Title

This action will delete the selected title from the database along with all related records that have foreign keys tied to that title. Additionally, all associated files will be deleted from the filesystem.

Warning: This action is dangerous and should be performed with caution!

  1. Navigate to the Movies section.
  2. Select the action "Delete movie and all its files".
  3. Check the title you wish to delete.
  4. Click on Go.
